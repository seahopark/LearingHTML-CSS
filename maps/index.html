1. info

  • minimumClusterSize: 클러스터 안에 들어가는 피커의 최소 갯수
  • Cluster Grid size: 동그란 클러스터를 구성하는 사각형의 크기
  • Data Count: 들어가는 데이터의 총 수
  • Max zoom level: 최고 확대 할 수 있는 레벨

앞에 두가지 세그먼트로 클러스터가 어떻게 보여질 것인지 결정될 듯.
그리드 사이즈가 작아지고 미니멈 클러스터 수가 커질수록 픽커가 더 많이 생기고, 클러스터는 줄어듬
즉, 미니멈 클러스터와 픽커 수는 정비례, 그리드 사이즈와 픽커는 반비례함.
클러스터 사이즈는 조금 씩 더 줄이고, 픽커가 많아질수록 조도 + 투명도를 조금씩 변경하는게 필요할 듯
